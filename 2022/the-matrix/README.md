# The Matrix

## Problem Statement

Return the inverse of a 2x2 matrix. If it doesn't exist, return the zero matrix.

## Input Format

- Each line represents a row of integers in the matrix A

## Constraints

$
rank(A) = 2
$


## Output Format
2d float array (2-decimal-places): inverse of matrix A if it exists

## Sample Input

```
5 -71
-1 4
```

## Sample Output

```
-0.08 -1.39
-0.02 -0.1
```
